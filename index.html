pelicanでブログカードを使いたい、それembedly-cardで出来るよ
Sun 12 March 2017

ブログを書いてると、リンクはきれいに表示したいと言うのが、人の常だと思います。

はてななんかだと、はてな記法で簡単にブログカードが出来るし、wordpressでも同じ。

これを使えば、pelicanでも簡単に出来るみたいだけど、それはそれ markdown を使っているのが前提。

こちらによれば、chrome拡張を使えば生のhtmlを発行してくれるみたいだけど、ちょっと面倒。

で、pelicanでrestructuredtextでブログカードとなれば、これ使うしかない。

ただ、pelicanの3.7.1では、エラーが出てうまく行かなかったので、以下の通り書き換えた。

--- a/embedly_cards/card_rst.py
+++ b/embedly_cards/card_rst.py
@@ -48,5 +48,5 @@ def content_object_init(instance):
     directives.register_directive('embedly-card', EmbedlyCard)

 def register():
-    signals.content_object_init.connect(content_object_init)
+    signals.initialized.connect(content_object_init)

書き換えたら、後は

python setup.py install

で、インストール。 あとは、こんなふうに書くだけ。

.. embedly-card:: http://www.yahoo.co.jp
